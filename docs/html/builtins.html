Prompt

Synopsis
setenv PROMPT "myprompt$>"

Variable of prompt

%n -> $USER
%p -> $PWD
%h -> Home pwd
%P -> Special PWD
%g -> $GROUP
%G -> Branch of Git
%t -> Current Time

Style of prompt

%u -> underline
%s -> surline

Color

%c{NAME_OF_COLOR}
Disponibilty:

  • GRAY
  • RED
  • GRE -> green
  • YEL -> yellow
  • BLUE
  • MAG -> Magenta
  • SBLUE -> blue sky
  • DEF -> default
Example:
setenv PROMPT "%c{RED}(%n)%c{BLUE}%u[%h]%e%c{DEF}$> "
setenv PROMPT "%c{BLUE}(%n)%c{GRE}%u{%P}%e\n%c{MAG}[%t]-%G->"
